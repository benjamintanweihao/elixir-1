# SessionStore

This is an Elixir implementation of the Mnesia application used to store
sessions from *Erlang and OTP in Action*.

# Starting Mnesia From IEx

    iex --erl "-mnesia dir '/tmp/mnesa_store'" --sname mnode -S mix
